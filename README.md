# DeepRead Pro

AI-powered Chrome extension for active English learning.

DeepRead Pro helps users learn English actively while browsing by combining translation, AI explanations, and vocabulary collection into a lightweight workflow.

## Features

* Highlight and translate selected text
* AI-powered explanation and contextual learning
* Vocabulary collection and review
* Active learning workflow inside browser
* Side panel interaction

## Tech Stack

* React
* Vite
* Chrome Extension API

## Installation

```bash
npm install
npm run dev
```
